Per D-01, D-02, D-03, D-04 — add three new commands to detent-tools.cjs:

**1. `cmdTruthPropose(targetDir, named)`:**
- Required args: `--id`, `--file` (e.g., "frozen-decisions", "constraint-ledger", "domain-model")
- Optional args: `--source-agent` (default "unknown"), `--rationale` (default ""), `--status` (default "PROPOSED"), `--retained-goal` (default ""), `--discarded-options` (default "")
- Resolve file path: `path.join(detentDir(targetDir), 'truth-surface', file + '.md')`
- If file doesn't exist, initialize with `# ${file}\n\n`
- Scan content for `## ${id}` header — if found, exit 1 with "Entry ${id} already exists"
- Append entry block in this exact format (CRITICAL — this is the constraint-ledger entry schema per TRUTH-03):
```
\n## ${id}\n\n\`\`\`yaml\nid: ${id}\nstatus: ${status}\nsource_agent: ${sourceAgent}\nchallenged_by: null\nfrozen_at: null\nretained_goal: "${retainedGoal}"\ndiscarded_options: "${discardedOptions}"\n\`\`\`\n\n${rationale}\n
```
- The `retained_goal` and `discarded_options` fields are always present in every entry (for schema consistency). For frozen-decisions and domain-model files, agents simply pass empty strings. For constraint-ledger entries, agents MUST populate these per TRUTH-03.
- Write atomically via `writeFileAtomicSync(filePath, content, { encoding: 'utf8' })`
- Output `{ "ok": true, "id": "<id>" }` JSON to stdout

**2. `cmdTruthFreeze(targetDir, named)`:**
- Required args: `--id`, `--file`
- Resolve file path same as propose
- If file doesn't exist → exit 1 "File ${file}.md not found"
- If `## ${id}` header not found → exit 1 "Entry ${id} not found. Call truth-propose first."
- Parse status from YAML block: regex `## ${id}[\s\S]*?status: (\w+)` — if "FROZEN" → exit 1 "Entry ${id} is already FROZEN (immutable)."
- Parse challenged_by: regex `## ${id}[\s\S]*?challenged_by: (\S+)` — if null → exit 1 "Entry ${id} is not mature (missing challenged_by). Cannot freeze."
- Replace `status: PROPOSED` with `status: FROZEN` in the entry's YAML block
- Replace `frozen_at: null` with `frozen_at: ${new Date().toISOString()}`
- Write atomically
- Output `{ "ok": true, "id": "<id>", "status": "FROZEN" }` to stdout

**3. `cmdTruthRead(targetDir, named)`:**
- Required arg: `--file`
- Resolve file path same as propose
- If file doesn't exist → exit 1 "File ${file}.md not found"
- Read file and write contents to stdout verbatim
