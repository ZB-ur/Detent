    1. **Check prerequisites:**

         Default if unclear: balanced
       - **Locale:** "Select locale for output language:\n- **en** — English\n- **zh-CN** — Chinese (Simplified)\n\nEnter: en or zh-CN"
         Default if unclear: en
       - **Unit granularity:** "Select coding unit granularity:\n- **fine** — small units, more iterations\n- **standard** (recommended) — balanced unit size\n- **coarse** — large units, fewer iterations\n\nEnter: fine, standard, or coarse"
         Default if unclear: standard

    4. **Run setup command:**
       - Execute via Bash: `node ./detent-tools.cjs setup --dir . --mode <mode> --budget <budget> --locale <locale> --granularity <granularity>`
       - Capture and verify the JSON output contains `"ok": true`

    5. **Verify and summarize:**
       - Read `.detent/config.json` via `node ./detent-tools.cjs config-read --dir .`
       - Print a summary showing:
         - Created files and directories
         - Configuration values chosen
         - Next step: "Run `/detent:discovery` to begin your first pipeline"

    **CRITICAL RULES (include verbatim in skill body):**
    - NEVER use the Write tool on any file inside `.detent/` — all writes go through detent-tools.cjs
    - If detent-tools.cjs exits non-zero, report the error and stop — do not attempt manual recovery
    - Validate user input for each question — if the answer doesn't match expected values, ask again
